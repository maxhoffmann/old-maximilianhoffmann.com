Reponsive Design

Since mobile web usage will soon surpass desktop’s internet traffic, websites have to be responsive. Making responsive websites is not easy as one has to change the workflow and there isn’t always a technical solution to every problem, but it also has advantages. The focus is shifted towards content and typography, because these are the best tools to enable a unified experience on all devices. I design in the browser, build SVG assets with Sketch and use responsive units for fluid interfaces.

dashboard
Performance

Mobile data plans are expensive and most often people have bad internet connections on the go as LTE, HSDPA or 3G isn’t available outside of cities. Additionally mobile phones have slow CPUs. Therefore I dig into the developer tools looking for high latencies, unnecessary requests, big paint times or “jank busters”. Using the GPU via CSS Transitions or Animations makes smooth 60fps possible even on mobile devices.
